# Flutter Chat AI

Flutter Chat AI is a minor Project to create a mobile app to represent ChatGPT, using OpenAI API

## Installation

Clone the repository:

```bash
git clone https://github.com/Erel0251/FlutterChatAI.git
```

Add Packages:

```bash
flutter pub add
```
